# Linux Command

| Command | Description                                 |
| ------- | ------------------------------------------- |
| du -sh  | 查看文件夹的大小，s表示summrize，仅显示总计 |
| df -h   | 查看磁盘空间的使用情况                      |
| free -h | 查看内存大小                                |
| top     | 查看内存使用情况                            |
